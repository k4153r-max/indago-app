# Indago App

Prototipo de app móvil (PWA) para **Indago.cl** — el comparador de precios de Chile.

## Características

- 🔍 Búsqueda de productos
- 📈 Historial de precios (gráfico)
- 🔔 Alertas de precio
- ❤️ Favoritos
- 📱 Diseño mobile-first + PWA (instalable en Android e iOS)
- 🇨🇱 Precios en CLP

## Cómo correr

```bash
npm install
npm run dev
```

Abre http://localhost:3000

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Recharts (gráficos)
- Lucide icons

## Nota

Esta es una **demo con datos mock**. Para producción se necesita conectar a la API real de Indago o un backend de scraping/aggregation.
